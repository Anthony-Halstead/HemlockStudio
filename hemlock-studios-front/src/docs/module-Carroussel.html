Carroussel Component. Displays a 3D carousel of items. The carousel's appearance and functionality can be customized through props.
Parameters:
Name Type Description
props Object Component properties.
Properties
Name Type Description
cards Array.<Object> An array of card objects to be displayed in the carousel. Each card object should have attributes required by the `Carousel` component.
offset number The offset radius for the carousel, determines how much the carousel is "curved".
showArrows boolean Determines if the carousel's navigation arrows should be shown.
width string CSS width property for the carousel.
height string CSS height property for the carousel.
margin string CSS margin property for the carousel.
